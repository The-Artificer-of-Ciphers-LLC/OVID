     ALLOWED_TRANSITIONS: dict[str, set[str]] = {

     def _validate_status_transition(current: str, target: str) -> None:

         allowed = ALLOWED_TRANSITIONS.get(current, set())

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Mastodon instance -> OVID API | External Mastodon instance could be malicious |
| User input -> disc status | Authenticated user attempts invalid state transitions |
| Provider error -> API response | External OAuth provider error text could contain secrets |
| Disc submission -> database | Concurrent submissions could cause integrity violations |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-07 | Spoofing | Mastodon DNS rebinding | mitigate | Pin resolved IP in `validate_mastodon_domain()`, check against private ranges (SEC-02, D-10) |
| T-1-08 | Spoofing | Mastodon blocked instances | mitigate | Hardcoded `_BLOCKED_INSTANCES` frozenset checked before DNS resolution (D-10) |
| T-1-09 | Denial of Service | Mastodon domain registration flood | mitigate | Rate limit new domain registrations to 3/hour per IP (D-10) |
| T-1-10 | Tampering | Disc status invalid transition | mitigate | `ALLOWED_TRANSITIONS` state machine enforced before status update (BUG-02, D-21) |
| T-1-11 | Information Disclosure | OAuth provider error in response | mitigate | Replace `str(e)` in error details with static codes, log detail server-side only (SEC-05, D-23) |
| T-1-12 | Tampering | Concurrent Mastodon client registration | mitigate | PostgreSQL `ON CONFLICT DO NOTHING` upsert (BUG-03, D-13) |
| T-1-13 | Spoofing | Mastodon email collision | mitigate | Include domain in placeholder email format (BUG-01, D-12) |
</threat_model>
