Name input: Uses project `inputClass` pattern (`rounded border border-neutral-300 bg-white px-3 py-2 text-sm outline-none focus:border-blue-500 focus:ring-1 focus:ring-blue-500 dark:border-neutral-700 dark:bg-neutral-900`), `className="flex-1"`, `placeholder="Chapter name (optional)"`, `maxLength={200}`, `aria-label={`Name for chapter ${ch.chapter_index}`}`, `data-testid={`chapter-name-${titleIndex}-${ch.chapter_index}`}`
